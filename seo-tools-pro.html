Centro esportazione file

Manifest SEO JSON Raccoglie i dati correnti della pagina di lavoro in un file unico archiviabile.
Snippet metadati Esporta il blocco meta pronto per essere incollato nella pagina reale.
Schema JSON-LD Esporta il file dati strutturati corrente in formato JSON.
Sitemap e robots Genera e scarica i file tecnici principali del sito.

I file vengono creati nel browser e scaricati localmente. Dopo la pubblicazione del sito, l’uso corretto consiste nel generare questi file, integrarli nelle pagine reali e aggiornare periodicamente sitemap e metadati.
