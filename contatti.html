Informazioni operative

Questa pagina è progettata per un sito statico pubblicato su GitHub Pages. Per un invio completamente automatico senza client email è necessario collegare il modulo a un servizio esterno dedicato oppure a un backend separato. 2

La struttura qui adottata mantiene un’impostazione corretta anche senza backend: validazione dei campi, protezione antispam di base tramite honeypot e invio tramite email precompilata.

  • Email ufficiale integrata nel markup semantico.
  • Modulo compatibile con GitHub Pages.
  • Protezione antispam di base senza esporre l’indirizzo in chiaro nel codice statico iniziale.
  • Struttura pronta a essere collegata in seguito a un servizio form dedicato.
